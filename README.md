# Sales Performance & Customer Behavior Analysis

## Business Scenario
I have recently been hired as a Junior Data Analyst at a retail company that operates multiple stores. The management team wants to better understand their sales performance, including which products drive revenue, how each store performs, and what kind of customers contribute the most to the business.

However, the company currently only looks at total sales without deeper insights into product contribution, store efficiency, and customer value.

## Objective
The goal of this project is to:
- Analyze revenue trends over time
- Identify top-performing and low-performing products
- Evaluate store performance and efficiency
- Understand customer purchasing behavior

## Dataset
The dataset consists of four tables:
- **Sales**: transaction data (quantity, price, date)
- **Products**: product details (name, category)
- **Stores**: store information (location)
- **Customers**: customer demographics (gender, age)

## Tools Used
- SQL (MySQL)
- Power BI

## Analysis Steps
1. Data cleaning and validation  
2. Data joining across multiple tables  
3. Revenue calculation  
4. Product performance analysis  
5. Store performance analysis  
6. Customer behavior analysis
